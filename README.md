## 使用方法
以下の順番で実行してください．
1. MyServer2を実行
2. 自身と相手のクライアントを作成するため，MyClientを2回実行
3. 実行時，IPアドレスを尋ねられるが何も入力しなくてよい．またユーザ名を尋ねられるが自身と相手で重複しないように入力
